The Input Fuction

input and print are examples of what we would call built-in functions-- funtions that are built into python. In other words, these functions come with python when you install it.

Input allows the program to receive input from the user and do something with that response.

Example:

  • This code will ask the user their name then print it with "greetings"
    name = input("whats your name?)
    print("greetings")

The int function

This takes the string returned by the input function, and passes it to the int function. The int function takes that string and converts it to an integer.



example:
n = int(input("Please enter a number:"))
Please enter a number: 4
n + 2

This will turn the value that we stored in "n" into an integer. If the "int" function was not used, "n" would be recognized as a string and throw an error.

The Len Function

The len function counts the number of characters in a string.

example:
word = "python"
len(word)
This line of code will give you the integer "6" because there are 6 characters in the word "python"

The len funtcion can also count the number of items in a string.

example:
len([3, 4, 5])
This will give you the integer "3", since there are 3 items in the list.

While Loops

A while loop will continue to run while some condition is true. As soon as the condition is false, the loop will stop.

Example:

  • n = 1
    while n < 3:
    ___print(n)
    ___n += 1

Before the loop, we set 'n' to a starting number: n = 1

Then we check is 'n' is less than 3: while n < 3:

The current value if 'n' is 1, so this means that 'n < 3' evaluates to True.
so the code inside the while loop will run.

Inside the loop, we first print the value of 'n', and then we increase 'n' by 1: n += 1

Then, we go back to the top of the loop and do the while thing over again. And we keep repeating this process until the value of 'n' gets large enough that 'n < 3' evaluates to False--at which point the loop stops running.
