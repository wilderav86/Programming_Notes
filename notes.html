Markup

-markup is plaintext that has a special meaning. Tags tell the browser how to display text. e.g. "< em >markup< /em >"

-sub element does this.

-sup element does this.

-useful for math and chemistry i.e. x3+2x2 or H2O

Lists

-Lists can be implemented using the < li > tag as ordered lists < ol >, or unordered lists < ul>.

the li tag must be nested in a ul or ol tag.

example:
An unordered list of groceries.

  • bananas
  • milk
  • cereal

An ordered list of the best ice cream.
  1. cookie dough
  2. choc chip
  3. mint

ordered lists can be nested within each other as well:

  • Mammals
    1. Racoons
    2. Gorillas

Linking Images:

Linking an image requires:

  1. < img src= > tag
  2. "image location"
  3. alt= tag
    • the alt tag tells the browser what to display if it can't find the image.

Documents: Head and Body

To publish a full HTML document you need:

  1. a < !DOCTYPE html > tag
    • This tells the browser that it is loading HTML
  2. an < html > tag
    • Html tags encompass all other tags
    • you can include "lang="en" to tell the browser to read with english language.
  3. a < head > tag. Head tags contain things like:
    • stylesheets in CSS
    • Script in javascript
    • metadata for search engines
  4. a < title > tag
    • the title tag is contained within the head tag
  5. a < body > tag
    • All other content is contained within the body tags
