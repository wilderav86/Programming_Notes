-----CSS-----

Styling

You can use inline styling to change the look of elements.

For instance, if i wanted to make everything in this < p > tag blue, I would add the inline tag: style="color: blue"; inside the opening P tag. This method can be repititious and is prone to errors.

You can add as many style attributes as you want as long as they are divided by a semicolon and inside the quotation marks.

Example: style="color: blue; text-align: center";

A more efficient way of styling is using Style elements

Style elements are used with selectors to style multiple elements at a time.

For instance, if you want to style all h2 elements with the same font you would create an opening and closing "style " tag that contains "h2 {font-style:"comic sans"}"

Selectors

Class selectors work the same way style elements use selectors to style multiple elements at a time. Say you want to change the color of everything contained in a single < p > tag, while leaving other < p > tags alone. You can insert class="class_name" into the < p > tag, then create a < style > tag containing ".class_selector { color:'red'}"

id selectors work in the same way, but only when a single element needs styling. instead of a "." to reference an id selector, it uses a "#"

You can also combine selectors by using descendant selectors. For instance, if you want to style all < em > tags within an < li > tag, you would use li em {color: red;}
